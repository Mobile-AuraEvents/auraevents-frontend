# Aura Events

Aplicação para gestão e consulta de eventos, com interface mobile em React Native/Expo e API backend em Spring Boot.

## Visão Geral

O projeto organiza os principais cadastros e relacionamentos do ecossistema de eventos:

- Shows
- Artistas
- Casas de show
- Patrocinadores
- Veículos de imprensa
- Vínculos entre shows e suas entidades relacionadas

## Estrutura do Projeto

- `auraevents-frontend`: aplicativo mobile desenvolvido com React Native e Expo
- `auraevents-backend`: API REST desenvolvida com Spring Boot

## Funcionalidades

- Listagem de shows com detalhes vinculados
- Cadastro e visualização de artistas
- Cadastro e visualização de espaços/casas de show
- Cadastro e visualização de patrocinadores
- Cadastro e visualização de veículos de imprensa
- Tela inicial com visão geral dos dados

## Tecnologias

- React Native
- Expo
- TypeScript
- Spring Boot
- PostgreSQL

## Como Executar

### Backend

1. Acesse a pasta `auraevents-backend`
2. Configure o banco de dados PostgreSQL
3. Execute a aplicação Spring Boot

### Frontend

1. Acesse a pasta `auraevents-frontend`
2. Instale as dependências
3. Inicie o app com Expo

## Observações

- O frontend consome a API exposta pelo backend.
